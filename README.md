# PRODIGY_WD_01 - Demo Calculator

## 📌 Summary
This project is part of the PRODIGY Web Development Internship.  
The task involves building a **web-based calculator** inspired by the demo at [sCalc](https://dunizb.github.io/sCalc/).  

The calculator performs basic arithmetic operations — **addition, subtraction, multiplication, and division** — while handling both valid and invalid inputs. It demonstrates core concepts of **HTML, CSS, and JavaScript** such as DOM manipulation, event handling, and responsive design.

---

## 🧮 Demo Calculator Used
Reference: [sCalc Demo Calculator](https://dunizb.github.io/sCalc/)  
This demo served as the base model for implementing the calculator logic and interface.

---

## 🚀 Features
- Perform addition, subtraction, multiplication, division
- Handle decimals, positive and negative numbers
- Follow **BODMAS precedence** for chained operations
- Error handling for invalid inputs (e.g., non-numeric characters, division by zero)
- Clear/reset functionality
- Responsive and styled interface
